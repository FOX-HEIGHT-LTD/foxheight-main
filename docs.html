FoxHeight Documentation

Dark-mode first, with code highlighting for TypeScript, Python, and YAML.

Quickstart

# Install CLI
npm i -g @foxheight/cli

# Authenticate
fox login --azure

# Create a project
fox init my-app

API Reference

import { FoxCloud } from '@foxheight/sdk';

const fox = new FoxCloud({ apiKey: process.env.FOXCLOUD_API_KEY });

const run = await fox.ai.invoke({
  model: 'gpt-4o',
  tools: ['search', 'retrieve'],
  input: 'Classify support tickets by topic'
});

TypeScript SDK

import { createClient } from '@foxheight/sdk';

const client = createClient({ region: 'eastus' });
const { data } = await client.data.query('select * from metrics limit 10');

Python SDK

from foxheight import Client

client = Client(region='eastus')
res = client.ai.invoke(model='gpt-4o', input='Summarize logs')
print(res.text)
